
  # Cafe Ordering System

  This is a code bundle for Cafe Ordering System. The original project is available at https://www.figma.com/design/VfkA8BdpSpfod9SN4Gem7U/Cafe-Ordering-System.

  ## Running the code

  Run `npm install` to install the dependencies.

  Run `npm run dev` to start the development server.
  
  Run `npm test` for doing stress and smoke test.
